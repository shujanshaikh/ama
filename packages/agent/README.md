# AMA Agent Tools

This package contains the local tool implementations used by the CLI runtime.

## Included tools

- `bash`
- `batch`
- `deleteFile`
- `editFile`
- `glob`
- `grep`
- `listDirectory`
- `readFile`
- `stringReplace`

## Development

```bash
cd packages/agent
npm install
npm run typecheck
npm run test
```
